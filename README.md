# GCD 绘图与数据处理 — Streamlit Web App

这是一个将 GCD（恒电流充放电）数据处理与绘图封装为 Streamlit 网页的项目。

快速开始

1. 在本地运行：

```bash
cd "C:\\Users\\Lenovo\\Desktop\\工作文件"
python -m streamlit run gcd_streamlit_app.py
```

2. 生成依赖文件：已包含 `requirements.txt`。

部署到 Streamlit Cloud（推荐）

1. 将仓库推送到 GitHub：

```bash
git init
git add .
git commit -m "Initial commit: gcd streamlit app"
git branch -M main
git remote add origin https://github.com/<your-username>/<repo-name>.git
git push -u origin main
```

2. 登录 https://share.streamlit.io，选择你的 GitHub 仓库并部署（Streamlit Cloud 会自动使用 `requirements.txt`）。

替代部署选项：Hugging Face Spaces、Azure、Docker 等。
